先说实话：谁适合读这篇，谁不适合。

如果你符合下面任意一条，这篇值得往下读：

如果你符合下面任意一条，这篇可以跳过或收藏备用：

  • 你现有的网络方案完全够用，AI 工具从未被封过

诚实地说：一台 VPS 不会让你的工作效率翻倍。它解决的是两个具体问题——网络更稳、多一个随时在线的 AI 助理。是否值得，取决于这两个问题对你来说痛不痛。

VPS 是什么，为什么要买

VPS 就是租了一台在云端的电脑，有独立的公网 IP，7×24 小时开机。你可以远程登录上去，装软件、跑服务、存文件——和操作自己电脑的区别是，它永远不关机，而且有一个互联网上唯一、只属于你的 IP 地址。

两个核心用途：

第一个：自建网络代理

大多数人的海外 AI 工具访问方案是买机场或共享节点。几十上百人共用一个 IP。AI 服务的安全系统怎么判断？一个 IP 同时有几十个账号在请求 ChatGPT、Claude——这怎么看都不像正常个人用户。结果就是频繁弹验证码、API 限流、甚至封号。

自建节点的优势非常直接：你独享一个 IP。在所有 AI 服务眼里，你就是一个普通海外家庭用户。没有共享 IP 的「信用分」问题，不会因为别人用这个 IP 干了什么而被连带处理。而且自建的稳定性远好于机场——你一个人用的带宽，没人跟你抢。

第二个：跑一个 7×24 的 AI 助理

你电脑上的 AI 工具，关了就是关了。半夜想到一个需求，你不可能爬起来开电脑。

但把 AI Agent 跑在 VPS 上，接入微信或飞书后，它就是住在你手机里的助理。你在地铁上、排队中、凌晨失眠时——打开手机发过去就行。它记得上下文，不是每次对话从零开始。

你电脑上的 AI 工具像你工位旁边的同事——你上班他上班，你下班他下班。它能帮你干很深入的活，但你得在工位上。

VPS 上的 AI Agent 像一个住在家里的管家，24 小时在线，随时发手机使唤。他做不了那么深的事情，但他永远在。

这两个不是竞品，是互补。坐电脑前深度工作用前者。离开电脑之后的所有事——交给后者。

第二步：装 Hermes Agent

这一步是整个流程里唯一需要手动操作的步骤。不过如果你能复制粘贴命令行到终端，就足够了。

买好 VPS 之后，你会收到一个 IP 地址和 root 密码。在你自己电脑上打开终端（Mac 的「终端」或 Windows 的 PowerShell），输入：

ssh root@你的服务器IP

第一次登录会提示确认指纹，输入 yes 回车。然后输入密码（注意：Linux 输入密码时屏幕上没有任何显示，这是正常的，输完直接回车即可）。

登录成功后，你会看到一个类似 root@你的主机名:~# 的提示符。然后跑这条命令：

pip install hermes-agent

安装完成后，运行：

hermes setup

你会看到一个配置向导，按提示填写：

配置项说明
AI 模型 Provider选择你的模型供应商
API Key填入你的 API 密钥
消息平台选择微信或飞书等

填完后启动服务，就可以了。

第三步：接入 DeepSeek 模型

Hermes 支持多种模型供应商：OpenAI、Anthropic、DeepSeek、OpenRouter 等。以 DeepSeek 为例：

1. 注册并获取 API Key

去 DeepSeek 官网注册账号，进入开发者后台，创建一个 API Key。DeepSeek 的费率极低，个人日常使用一个月就几块钱。

2. 配置到 Hermes

在 hermes setup 的配置向导中选择 Provider 为 DeepSeek，填入你的 API Key。或者直接编辑配置文件：

hermes config set model.default deepseek-chat
hermes config set providers.deepseek.api_key 你的API_KEY

3. 设置模型

DeepSeek 的模型选择：

  • deepseek-chat（V3）：日常对话、写作、信息处理
  • deepseek-reasoner（R1）：复杂推理、代码 Debug、分析任务

可以根据使用场景切换，或者设置默认模型。

4. 控制费用

可以在 Hermes 设置每日/每月 Token 消耗上限，到了自动停。同时在 DeepSeek 后台设置消费限额，双保险。
